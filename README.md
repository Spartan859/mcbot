# mcbot

A Minecraft bot

## Install

```shell
npm install
```

## Usage

Create `.env` file:

```
MC_HOST=<mc server host>
MC_USERNAME=<username>
MC_AUTH=offline
```

Then start the bot:

```shell
npm start
```

Support commands in game:

- `/msg <bot> come`: Make the bot come to you.
- `/msg <bot> gotoxz <x> <z>`: Make the bot go to the coordinates.
